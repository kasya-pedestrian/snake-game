main.c　
アプリケーションの基本的な構造、メインメニューの流れ 
main()

game_win.c 
game_win.h　
ゲームに関する関数のライブラリ、Windowsしか実行できません。 
enum direction, 
ヘビの方向を指示する
enum gameobjects, 
ゲームボードに置いていること指示する
struct snake, 
ヘビに関する構造体。リングバッファとして配列にヘビの体と頭の座標が保存されている。頭から尻尾までの順位は頭のポインターから頭のポインターの次まで。
int check_collision(int *); 
「引数」プレイヤーのスコア
「機能」衝突を確認する関数、衝突がなければ、ヘビの頭は指示された方向に進む。
「戻り値」０＝衝突起こった、１＝衝突なし
void player_input(); 
「機能」ユーザー入力を処理する関数
void clear_board(); 
「機能」ゲームが終わると、マロックされたメモリーを解放する関数
void clear_screen(); 
「機能」画面にある文字を全部消す関数
void draw_screen(int); 
「引数」表示するスコア
「機能」スコア、ゲーム盤、ヘビの体、エサを画面に書く関数
void setup_board(unsigned char); 
「引数」ゲーム盤の幅を定義する値
「機能」遊べる前にゲームを設定する関数
void sleep(int); 
「引数」停止するミリ秒の値
「機能」設定されたミリ秒でプロデューサーを停止する関数

menus.c
menus.h　
ゲームのメニューに関する関数のライブラリ 
char main_menu();
「機能」最初のメニューを表示する関数、またユーザー入力を処理する関数
「戻り値」ユーザーが入力の値

void settings_menu(int*, int*);
「引数」ゲーム盤のサイズの値、ヘビの速度の値
「機能」設定のメニューを表示する関数、またユーザー入力を処理する関数。入力をチェックし、引数のポインターで入力を保存する。
void score_menu();
「機能」スコアを表示する関数
